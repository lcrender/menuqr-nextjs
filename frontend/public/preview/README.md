# Imágenes de vista previa de plantillas

Colocá aquí las fotos de portada, logos y preview de cada plantilla.

**Nombres de archivo** (reemplazá `{plantilla}` por: classic, minimalist, foodie, burgers, italianFood):

- **Portada:** `portada-{plantilla}.jpg`  
  Ejemplo: `portada-classic.jpg`, `portada-minimalist.jpg`
- **Logo:** `logo-{plantilla}.jpg`  
  Ejemplo: `logo-classic.jpg`, `logo-minimalist.jpg`
- **Preview (panel admin):** `preview-{plantilla}.jpg`  
  Ejemplo: `preview-classic.jpg`, `preview-foodie.jpg`. Si no existe, se usa `preview-default.svg`.

**Plantillas:** classic, minimalist, foodie, burgers, italianFood

Las imágenes se sirven en la ruta `/preview/` (ej.: `/preview/portada-classic.jpg`, `/preview/preview-classic.jpg`).
