# Seeds (datos de ejemplo)

Aquí van archivos de referencia para poblar o probar la aplicación: menús en CSV, más adelante restaurantes, etc.

- **`menus/`** — CSV listos para la función **Importar menú con CSV** del panel (mismo formato que `frontend/public/templates/menu-import-ejemplo.csv`).
- El menú y el restaurante se eligen en pantalla al importar; en el CSV solo hace falta **sección + producto + precios** (y columnas opcionales).

Ver documentación en la app: **Documentación** → sección **2B — Importar menú y productos con CSV**.
